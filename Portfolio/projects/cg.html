FIFO Page Replacement

  • Tech Stack: OpenGL,Visual studio
  • Github URL: Project Link

FIFO (First-In, First-Out) page replacement is a simple algorithm used in computer graphics to manage memory allocation. It replaces the oldest page when a new page needs to be loaded.

Each page is assigned a timestamp, and the page with the oldest timestamp is replaced first.

FIFO is easy to implement but can suffer from "Belady's Anomaly," where increasing memory frames may result in more page faults. Nevertheless, FIFO is commonly used in computer graphics systems.
